connect-mongo package stores MongoDB session for Express and Connect

So, implemented sessions using Passport, but for storing sessions into my mongodb I use connect-mongo using a mongoose connection to connect to the mongodb database.

Then most standard implementation code is given in the officila doc

const session = require('express-session');
const MongoStore = require('connect-mongo')(session);

app.use(session({
    secret: 'foo',
    store: new MongoStore({ mongooseConnection: connection })
}));

connect-mongo stores sessions in the “sessions” collection by default.
