# arr-exclude

> Exclude certain items from an array


## Install

```
$ npm install --save arr-exclude
```


## Usage

```js
const arrExclude = require('arr-exclude');

arrExclude(['a', 'b', 'c'], ['b']);
//=> ['a', 'c']
```
